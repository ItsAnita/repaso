Git comandos

Git es un software de control de versiones diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

  1. git init: inicializamos el proyecto con git, git le da seguimiento a nuestro proyecto
  2. git add: sirve para añadir archivos al stage area
  3. git commit: sube los cambios al repsositorio local
  4. git log --oneline: lista los commits realizados
